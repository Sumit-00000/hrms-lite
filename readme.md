# HRMS Lite

## Overview
HRMS Lite is a lightweight Human Resource Management System that allows an admin to manage employees and track daily attendance.

## Features
- Employee management (Add, View, Delete)
- Attendance tracking (Present / Absent)
- Dashboard summary
- Clean and professional UI

## Tech Stack
Frontend:
- React (Vite)
- Axios

Backend:
- Node.js
- Express
- MongoDB

## Live URLs
Frontend: https://hrms-lite-six-xi.vercel.app
Backend: https://hrms-lite-qnf9.onrender.com

## Setup (Local)
1. Clone repository
2. Install backend dependencies
3. Configure .env
4. Run backend and frontend

## Assumptions
- Single admin user
- No authentication
